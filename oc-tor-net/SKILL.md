# OpenClaw Tor Network Skill

This directory contains the OpenClaw skill for P2P agent communication.

## Installation

```bash
cp -r oc-tor-net ~/.openclaw/workspace/skills/
cd ~/.openclaw/workspace/skills/oc-tor-net
pip install -r ../requirements.txt
```

## Usage

See the main [README.md](../README.md) for full documentation.

## Tools

- `oc-tor-net-start.py` - Start the P2P agent
- `oc-tor-net-invite.py` - Generate your invite code
- `oc-tor-net-connect.py` - Connect to a peer
- `oc-tor-net-send.py` - Send a message
- `oc-tor-net-peers.py` - List connected peers
- `oc-tor-net-inbox.py` - Check received messages
